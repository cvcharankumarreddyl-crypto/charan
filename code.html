Understanding Variables
Variables are containers that store data values. In JavaScript, you can declare variables using let, const, or var.
let message = "Hello, World!"; const PI = 3.14159; var userName = "CodeMaster";
Best Practice: Use const for values that won't change, and let for values that will change.
